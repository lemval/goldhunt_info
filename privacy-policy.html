PRIVACY POLICY

This privacy policy governs your use of the software application named 'GoldHunt' in both the unpaid as well as paid version (further referred as "Application"). The Application was created by Lemval Software ("We", "Us") and is meant for the Geocaching activity by means of searching and showing geocaches of Groundspeak™, newly published or otherwise.

The Application can be found in the store under GoldHunt Pro (Geocaching) and under GoldHunt (Geocaching)

What information does the Application obtain and how is it used?

User Provided Information

The Application obtains the information you provide via consent when authenticating against Geocaching.com and thereby grant access to your profile. This includes your username, avatar, configured location and profile picture with the sole purpose of a correct working application.

Automatically Collected Information

In addition, the Application may collect certain information automatically, including, but not limited to, the type of mobile device you use, your mobile operating system, and information about the way you use the Application.
However, Automatically Collected Information is in no way used by Us or the Application other than by explicit opt-in user consent via manually triggered feedback by mail.

Which data is stored by the Application?

The Application stores personal information, including set preferences, not yet submitted geocache visit registrations, Groundspeak profile data, locations (also as part of application logging), lists and drafts from Geocaching.com, and user statistics from Geocaching.com.
The Application stores non-personal information for the purpose of the functionality, including map tiles, geocaches, custom waypoints, puzzle codes, and images associated with geocaches.

No data is stored by the Application on devices other than on which the Application is installed. Any data stored will, unless explicitly exported or transmitted by the user, be removed from the device upon clearing application data.

Does the Application collect precise real time location information of the device?

When you use the Application, We use GPS technology (or other similar technology) to determine your current location in order to (1) display your location on the map, to (2) search for new caches on Geocaching.com around your current location, and to (3) support navigation and routing to a geocache. We will not share your current location with other users or partners otherwise. If you do not want the Application to use your location for the purposes set forth above, you should not grant or revoke the location permission to the Application all together, or disable individual services in the application.

We will only share information with third parties in the ways that are described in this privacy statement. No advertisement services are used.

None of the User Provided Data or Automatically Collected Information is purposely shared with third parties, even for purposes of Analytics or Advertisement.

Which permissions does the Application use?

Notification of new geocaches

The notification process runs following your given profile during certain times. As you want to be notified as soon as events happen (namely the new publication of a geocache), a number of permissions are required.

Access to the user its location on the background is needed to be alerted of new publications in your vicinity. You do not care about new publications of geocaches if they are out of your reach. Therefore the Application collects the location, both while actively using the application as well as when you currently not using your phone, in order to let you know, based on your preferences, when and how you are notified.

To make sure you are notified timely, we use notifications, exact alarms and wake the device.

Visualizing and navigation

A fine location based on GPS signals supports showing geocaches on route, in the vicinity (on the map and in the list view) and allows you to navigate to a certain geocache.

For routes, the OpenStreetMap library is used. For navigation, the Application looks for available options and presents those to the user.

Basic functionality

For retrieving information from Groundspeak, Inc., the Application uses data traffic over the network.

To allow making backups, store images of geocaches, import or export GPX files, access to storage is requested. Not granting the permission will restrict the use of this functionality.
